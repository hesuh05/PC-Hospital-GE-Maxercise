# 🏥 PC-Hospital-UDN-Gerencia

## Descripción

Sistema integral de gestión hospitalaria para la Universidad Distrital Francisco José de Caldas. Integra múltiples módulos: APIs REST, aplicaciones web, aplicaciones wearable y bases de datos robustas.

## Contenido Esperado

```
PC-Hospital-UDN-Gerencia/
├── Docs/              - Documentación técnica y requisitos
└── DataModels/        - Modelos de Machine Learning
```

## Objetivo

Centralizar la gestión hospitalaria en una plataforma unificada, mejorando eficiencia operativa con soluciones innovadoras y acceso en tiempo real a información crítica.
