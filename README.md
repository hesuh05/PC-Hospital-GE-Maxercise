# 🏥 PC-Hospital-UDN-Gerencia

## Descripción

Sistema integral de gestión hospitalaria para la Universidad Tecnológica De Xicotepec de Juárez. Integra múltiples módulos: APIs REST, aplicaciones web, aplicaciones wearable y bases de datos robustas.

## Contenido Esperado

```
PC-Hospital-UDN-Gerencia/
├── Docs/              - Documentación técnica y requisitos
├── DataBases/         - Esquemas y datos relacionales/no relacionales
├── DataModels/        - Modelos de Machine Learning
└── Deliverables/      - Aplicaciones y entregables finales
```

## Objetivo

Centralizar la gestión hospitalaria en una plataforma unificada, mejorando eficiencia operativa con soluciones innovadoras y acceso en tiempo real a información crítica.
