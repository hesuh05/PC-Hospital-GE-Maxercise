# Graphical User Interfaces (Interfaces Gráficas)

Documentación de diseño y especificación de interfaces gráficas para el proyecto PC-Hospital-UDN-Gerencia.

## Descripción

Esta carpeta contiene el diseño, mockups, prototipos y especificaciones de las interfaces gráficas de usuario (UI) para todas las aplicaciones del sistema.

## Estructura

```
GUIs/
├── WearableApp/   - Interfaz para aplicación Wearable
├── WebApp/        - Interfaz para aplicación Web
└── README.md
```

## Contenido Incluido

Cada interfaz especifica:

- **Wireframes**: Estructura básica de pantallas
- **Mockups**: Diseño visual detallado
- **Prototipos**: Versiones interactivas
- **Guías de estilo**: Colores, tipografía, iconografía
- **Patrones de interacción**: Flujos de usuario
- **Especificaciones técnicas**: Dimensiones, responsive design
- **Accesibilidad**: Requisitos WCAG y AA

## Aplicaciones

### WearableApp
Interfaz optimizada para dispositivos wearable (relojes inteligentes, pulseras fitness, etc.)
- Pantallas pequeñas
- Interacciones simplificadas
- Notificaciones y alertas
- Monitoreo en tiempo real

### WebApp
Interfaz web completa para navegadores
- Dashboard principal
- Funcionalidades avanzadas
- Gestión de datos
- Reportes y análisis

## Principios de Diseño

- **User-Centered**: Basado en necesidades reales de usuarios
- **Usabilidad**: Interfaz intuitiva y fácil de aprender
- **Accesibilidad**: Diseño inclusivo para todos
- **Consistencia**: Elementos y patrones uniformes
- **Feedback**: Retroalimentación clara del sistema
- **Eficiencia**: Minimizar pasos para tareas frecuentes

## Audiencia

- Diseñadores UX/UI
- Desarrolladores frontend
- Product managers
- Usuarios finales
- Testers de UX

## Herramientas Utilizadas

- Figma, Adobe XD, Sketch
- Prototipado interactivo
- Testing de usabilidad
- Análisis de accesibilidad

## Relaciones

Los GUIs se basan en:
- Requisitos de Usuario (URs)
- Historias de Usuario (UHs)
- Requisitos Funcionales (FRs)

Y sirven para:
- Guiar desarrollo frontend
- Validar conceptos con usuarios
- Testing y QA
- Documentación de usuario final
