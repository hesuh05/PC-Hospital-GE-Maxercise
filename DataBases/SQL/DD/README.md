# Diccionario de Datos SQL

Diccionario de datos que describe las tablas, columnas y relaciones de las bases de datos SQL del proyecto. Incluye definiciones de tablas, claves y restricciones.

## Propósito

Proporcionar una referencia detallada de la estructura de datos SQL para apoyar el desarrollo de aplicaciones y asegurar la integridad de los datos.

## Formatos de Archivos

Aquí encontrarás archivos de documentación en formatos .md, .txt o .pdf que detallan las estructuras de tablas, tipos de datos y reglas de negocio.
