# Diccionario de Datos SQL

## Descripción

Diccionario de datos que describe las tablas, columnas y relaciones de las bases de datos SQL del proyecto. Incluye definiciones de tablas, claves y restricciones.

Proporciona una referencia detallada de la estructura de datos SQL para apoyar el desarrollo de aplicaciones y asegurar la integridad de los datos.

## Estructura del Directorio

Este directorio no contiene subniveles adicionales; se enfoca en documentación del diccionario de datos.
