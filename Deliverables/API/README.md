# API Backend

![Component](https://img.shields.io/badge/Component-API-orange) ![Role](https://img.shields.io/badge/Role-Backend%20Service-blue) ![Protocol](https://img.shields.io/badge/Protocol-HTTP%20%2F%20REST-green)

## Descripción

Este directorio contiene toda la documentación, código fuente y recursos relacionados con la API del proyecto PC-Hospital-GE-Maxercise, la cual actúa como el backend central del sistema para el procesamiento de datos y la comunicación con los clientes.

Proporciona una estructura organizada para separar la lógica de negocio, las configuraciones de construcción/compilación de artefactos y los manuales técnicos para el despliegue del servidor.
