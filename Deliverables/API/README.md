# API

## Descripción
Este directorio contiene la API principal del proyecto PC-Hospital-UDN-Gerencia. Reúne el servidor híbrido, la configuración de conexiones a bases de datos, los controladores, modelos, rutas y la documentación técnica necesaria para su ejecución.

## Estructura del Directorio

```
API/
├── build/
├── DeployManual/
├── src/
├── server.js
├── package.json
└── swagger.json
```

## Contenido esperado

Aquí se pueden almacenar:

- Configuración del servidor
- Controladores y rutas
- Modelos de datos
- Conexión a MySQL y MongoDB
- Documentación Swagger
- Manuales de despliegue
- Recursos de compilación o empaquetado

## Objetivo
Centralizar la lógica del backend que da soporte a los módulos de gerencia hospitalaria, incluyendo seguridad, consumo de datos y documentación de la API.
