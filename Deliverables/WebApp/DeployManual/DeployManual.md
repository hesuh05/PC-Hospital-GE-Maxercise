# DeployManual

## Descripción
Esta carpeta contiene la documentación técnica para el despliegue de la aplicación web.

## Contenido esperado
Aquí se pueden almacenar:

- Procedimientos de despliegue
- Configuración de hosting
- Variables de entorno
- Configuración de servidores web
- Configuración SSL
- Pasos de actualización
- Configuración de dominios

## Objetivo
Documentar el proceso de instalación y publicación de la aplicación web en distintos ambientes.
