# Aplicación Web

![Component](https://img.shields.io/badge/Component-Web%20App-blue)  ![Role](https://img.shields.io/badge/Role-Client%20%26%20Server-orange)  ![Interface](https://img.shields.io/badge/Interface-Responsive-green)

## Descripción

Este directorio contiene toda la documentación, código fuente y recursos relacionados con la aplicación web (WebApp) del proyecto PC-Hospital-GE-Maxercise. La aplicación abarca tanto los componentes visuales de cara al usuario final (frontend) como la lógica y servicios de soporte (backend).

Proporciona una estructura organizada para separar el código fuente de frontend y backend, así como los manuales para usuarios finales y técnicos de instalación.

## Estructura del Directorio

```text
WebApp/
├── DeployManual/
│   └── DeployManual.md
├── UserManual/
│   └── UserManual.md
└── source/
    ├── backend/
    │   └── backend.md
    └── frontend/
        └── frontend.md
```
