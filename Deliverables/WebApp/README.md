# Web App

## Descripción
Este directorio contiene los entregables de la aplicación web del proyecto PC-Hospital-UDN-Gerencia. Agrupa los recursos de despliegue, documentación de usuario y el código fuente dividido en backend y frontend.

## Estructura del Directorio

```
WebApp/
├── DeployManual/
├── source/
│   ├── backend/
│   └── frontend/
└── UserManual/
```

## Contenido esperado

Aquí se pueden almacenar:

- Manuales de despliegue
- Manuales de usuario
- Código fuente backend
- Código fuente frontend
- Recursos de integración con servicios externos
- Documentación técnica del proyecto web

## Objetivo
Organizar los componentes necesarios para construir, desplegar y documentar la aplicación web del sistema.
