# Backend

## Descripción
Esta carpeta contiene el código fuente backend de la aplicación web. La referencia técnica complementaria está en [backend.md](backend.md).

## Contenido esperado

Aquí se pueden almacenar:

- APIs
- Servicios
- Lógica de negocio
- Acceso a base de datos
- Middleware
- Validaciones
- Autenticación y autorización
- Integraciones externas

## Objetivo
Gestionar toda la lógica del servidor y procesamiento de información de la aplicación web.
