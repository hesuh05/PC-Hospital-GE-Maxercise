# Source

## Descripción
Esta carpeta contiene el código fuente de la aplicación web. Se organiza en dos capas principales: backend y frontend.

## Estructura del Directorio

```
source/
├── backend/
└── frontend/
```

## Contenido esperado

Aquí se pueden almacenar:

- Lógica de negocio
- Interfaces y vistas
- Servicios y APIs
- Configuración de comunicación entre capas
- Utilidades compartidas

## Objetivo
Centralizar la implementación funcional de la aplicación web para facilitar su mantenimiento y evolución.
