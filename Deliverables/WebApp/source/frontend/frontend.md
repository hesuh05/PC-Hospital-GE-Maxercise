# Frontend Web

![Frontend](https://img.shields.io/badge/Frontend-HTML5%20%2F%20CSS3-orange) ![Framework](https://img.shields.io/badge/Framework-React-blue) ![Service](https://img.shields.io/badge/Service-API%20Consumption-green)

## Descripción

Esta carpeta contiene el código fuente frontend de la aplicación web.

Implementa la interfaz gráfica y experiencia de usuario de la aplicación web, incluyendo interfaces de usuario, componentes visuales, vistas, hojas de estilo CSS y el consumo de las APIs del sistema.

## Estructura del Directorio

Este directorio no contiene subniveles adicionales; se enfoca en la interfaz gráfica y experiencia visual de la aplicación web.
