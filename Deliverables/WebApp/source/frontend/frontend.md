# Frontend

## Descripción
Esta carpeta contiene el código fuente frontend de la aplicación web.

## Contenido esperado
Aquí se pueden almacenar:

- Interfaces de usuario
- Componentes visuales
- Vistas y páginas
- Estilos CSS
- Configuración de rutas
- Consumo de APIs
- Recursos multimedia

## Objetivo
Implementar la interfaz gráfica y experiencia de usuario de la aplicación web.
