# UserManual

## Descripción
Esta carpeta contiene la documentación orientada al usuario final del sistema.

## Contenido esperado
Aquí se pueden almacenar:

- Manuales de usuario
- Guías de uso
- Tutoriales
- Procedimientos operativos
- Preguntas frecuentes
- Capturas de pantalla
- Buenas prácticas

## Objetivo
Facilitar el aprendizaje y uso correcto del sistema para usuarios finales.
