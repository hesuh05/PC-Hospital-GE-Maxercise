# Wearable App

## Descripción
Este directorio contiene los entregables de la aplicación wearable del proyecto PC-Hospital-UDN-Gerencia. Agrupa los recursos de compilación, los manuales de despliegue y usuario, además del código fuente dividido en backend y frontend.

## Estructura del Directorio

```
WearableApp/
├── build/
├── DeployManual/
├── source/
│   ├── backend/
│   └── frontend/
└── UserManual/
```

## Contenido esperado

Aquí se pueden almacenar:

- Archivos de build y empaquetado
- Manuales de instalación y despliegue
- Documentación de usuario final
- Código fuente backend
- Código fuente frontend
- Recursos de integración con dispositivos wearable

## Objetivo
Organizar todos los materiales necesarios para construir, desplegar y documentar la aplicación wearable del sistema.
