# Aplicación Wearable

![Component](https://img.shields.io/badge/Component-Wearable%20App-green)  ![Platform](https://img.shields.io/badge/Platform-WearOS%20%2F%20Android-blue)  ![Role](https://img.shields.io/badge/Role-Client%20App-orange)

## Descripción

Este directorio contiene toda la documentación, código fuente y recursos relacionados con la aplicación wearable (WearableApp) del proyecto PC-Hospital-GE-Maxercise. La aplicación está diseñada para su ejecución en dispositivos compatibles, administrando la interfaz de usuario, sensores y comunicación del lado servidor.

Proporciona una estructura organizada para separar el código fuente de frontend y backend, la configuración de compilación de la app, así como los manuales para usuarios finales y técnicos de instalación.
