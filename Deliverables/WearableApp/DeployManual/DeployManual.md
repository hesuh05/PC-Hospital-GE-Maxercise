# DeployManual

## Descripción
Esta carpeta contiene la documentación necesaria para instalar y desplegar la aplicación wearable en dispositivos compatibles.

## Contenido esperado
Aquí se pueden almacenar:

- Manuales de instalación
- Configuración inicial
- Requisitos del dispositivo
- Procedimientos de actualización
- Configuración de conexión con backend
- Pasos de publicación

## Objetivo
Proporcionar instrucciones claras para el despliegue correcto de la aplicación wearable.
