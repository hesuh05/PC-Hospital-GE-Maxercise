## Descripción

Esta carpeta contiene la documentación necesaria para instalar y desplegar la aplicación wearable en dispositivos compatibles.

Proporciona instrucciones claras para el despliegue correcto de la aplicación wearable, incluyendo manuales de instalación, configuraciones de conexión con backend y requisitos de dispositivos.

## Estructura del Directorio

Este directorio no contiene subniveles adicionales; se enfoca en la documentación para la instalación y despliegue de la aplicación wearable.
