# Frontend

## Descripción
Esta carpeta contiene la interfaz y componentes visuales de la aplicación wearable. La documentación complementaria de esta sección está en [frontend.md](frontend.md).

## Contenido esperado

Aquí se pueden almacenar:

- Pantallas de usuario
- Componentes visuales
- Navegación
- Recursos gráficos
- Estilos
- Configuración de interfaz
- Integración con sensores o dispositivos

## Objetivo
Gestionar la experiencia visual e interacción del usuario dentro de la aplicación wearable.
