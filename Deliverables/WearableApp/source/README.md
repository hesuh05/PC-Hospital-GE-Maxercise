# Source

## Descripción
Esta carpeta contiene el código fuente principal de la aplicación wearable. Se organiza en dos ramas: backend y frontend.

## Estructura del Directorio

```
source/
├── backend/
└── frontend/
```

## Contenido esperado

Aquí se pueden almacenar:

- Lógica del backend
- Interfaces y pantallas del frontend
- Servicios de integración
- Configuración de comunicación entre capas
- Utilidades compartidas

## Objetivo
Centralizar la implementación funcional de la aplicación wearable para facilitar su mantenimiento y evolución.
