# Backend Wearable

![Service](https://img.shields.io/badge/Service-API%20REST-orange) ![Protocol](https://img.shields.io/badge/Protocol-JSON-blue) ![Role](https://img.shields.io/badge/Role-Data%20Sync-green)

## Descripción

Esta carpeta contiene los componentes backend utilizados por la aplicación wearable para comunicación, procesamiento y sincronización de datos.

Administra la lógica del lado servidor necesaria para el funcionamiento de la aplicación wearable, incluyendo APIs de soporte, servicios de autenticación y gestores de integración.

## Estructura del Directorio

Este directorio no contiene subniveles adicionales; se enfoca en la lógica del lado servidor de la aplicación wearable.
