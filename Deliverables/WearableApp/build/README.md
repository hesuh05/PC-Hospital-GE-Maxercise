# Build

## Descripción
Esta carpeta contiene los recursos relacionados con la compilación y empaquetado de la aplicación wearable. La documentación detallada de esta sección se encuentra en [build.md](build.md).

## Contenido esperado

Aquí se pueden almacenar:

- Archivos compilados
- Scripts de construcción
- Configuraciones de build
- Paquetes generados
- Recursos temporales de compilación

## Objetivo
Gestionar todos los elementos necesarios para generar versiones ejecutables de la aplicación wearable.
