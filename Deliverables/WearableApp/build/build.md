# Build

## Descripción
Esta carpeta contiene los recursos relacionados con la compilación y empaquetado de la aplicación wearable.

## Contenido esperado
Aquí se pueden almacenar:

- Archivos compilados
- Scripts de construcción
- Configuraciones de build
- APKs o paquetes generados
- Configuración de automatización
- Recursos temporales de compilación

## Objetivo
Gestionar todos los elementos necesarios para generar versiones ejecutables de la aplicación wearable.
