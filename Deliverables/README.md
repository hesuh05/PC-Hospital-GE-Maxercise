# Entregables

![API](https://img.shields.io/badge/Deliverable-API-orange) ![Wearable App](https://img.shields.io/badge/Deliverable-Wearable%20App-green) ![Web App](https://img.shields.io/badge/Deliverable-Web%20App-blue)

## Descripción

Este directorio contiene todos los entregables relacionados con el proyecto PC-Hospital-GE-Maxercise, organizados por componente (API, WearableApp y WebApp).

Proporciona una estructura organizada para los entregables del sistema, facilitando el acceso a códigos fuente, manuales de usuario, manuales de despliegue y compilaciones.

## Estructura del Directorio

```text
Deliverables/
├── API/
│   ├── DeployManual/
│   ├── build/
│   └── source/
├── WearableApp/
│   ├── DeployManual/
│   ├── UserManual/
│   ├── build/
│   └── source/
│       ├── backend/
│       └── frontend/
└── WebApp/
    ├── DeployManual/
    ├── UserManual/
    └── source/
        ├── backend/
        └── frontend/
```
